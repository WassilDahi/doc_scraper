# doc_scraper


## How to use :

### Install requirements :
```shell
pip install -r requirements.txt
```

### Enter links in the file urls.txt : 
- Must be one link per line.
- Every link must begin with http/https.

### Execute :
```shell
python doc_scraper.py
```
- tested only on python 3.
